mds-sale-line-twocolumn
=======================
Tryton modules to add a two-column-line-type to sales.

How To
======
You need to update your sale-templates. 
A sample template is included in the module folder.

Install
=======

pip install mds-sale-line-twocolumn

Requires
========
- Tryton 5.0
